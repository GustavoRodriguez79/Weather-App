# Weather App ☀️🌧️  
Aplicación que muestra el clima usando la API de OpenWeatherMap.  

## 🚀 Características  
- Búsqueda por ciudad.  
- Muestra temperatura, condiciones climáticas e icono descriptivo.  

## 🛠️ Tecnologías  
- HTML  
- CSS  
- JavaScript  
- API: [OpenWeatherMap](https://openweathermap.org)
